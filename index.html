NTU Timetable Optimiser

Options

These options persist between reloads.

Free day bonus

Increase if you really value free days.

This is added per free day. Odd weekdays and even weekdays are considered separately.

Lunch break bonus

Increase if you really value lunch breaks.

This is added per day if there is a period of at least 2 slots with no lessons that falls completely between 1100 and 1400. Odd weekdays and even weekdays are considered separately.

Slot penalties

If you don't like waking up early, you could add a penalty for the first few slots of each day. If you want each day to end earlier, you could add a penalty for the last few slots for each day. If you want to go home early on Friday, you could add a penalty for the later slots on Friday.

These are subtracted if there is a class at the particular time, once for even weeks and once for odd weeks.
