Options

Free day bonus

Increase if you really value free days.

This is added per free day per week.

Lunch break bonus

Increase if you really value lunch breaks.

Lunch should fall between slots and , inclusive.

Lunch should be at least slot(s) long.

This is added per day per week if there is a period of at least the specified number of slots with no lessons that falls completely between the specified times.

Slot penalties

If you don't like waking up early, you could add a penalty for the first few slots of each day. If you want each day to end earlier, you could add a penalty for the last few slots for each day. If you want to go home early on Friday, you could add a penalty for the later slots on Friday.

These are subtracted if there is a class at the particular day and time, per week. You can set these to negative values to make them slot bonuses.
